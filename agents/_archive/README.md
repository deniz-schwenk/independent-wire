# Archived agent prompts

Prompts no longer loaded by any pipeline. Kept because git history
alone is not enough — keeping the file makes prior versions easy to
consult without checking out commits.

Files here are dead. Do not edit.

Filename convention: `<source-dir>-<source-file>-YYYY-MM-DD.md`. The
date is the day the file was archived.

The authoritative inventory of every prompt — active, disabled,
archived, and unresolved — lives in `docs/AGENT-INVENTORY.md`.
